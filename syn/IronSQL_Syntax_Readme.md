# IronSQL 语法标准详情说明

- IronSQL 是基于现代 C++ 开发的个人轻量数据库系统
  * IronSQL 的语法标准文件在目录`.../IronSQL/syn/IronSQL_Syntax_zh_CN.txt`.
  * 需要注意,目前这个项目只有本人开发与维护,所以项目的进度极其缓慢.
  * 当你运行 IronSQL 二进制程序的时候发现,使用 IronSQL 标准语法说明文件语法执行 SQL 语句时候,
    请使用帮助命令'help -syntax --all;'.
  * 本人保证:
    * IronSQL 二进制程序的实际操作语法,会以 IronSQL 的标准语法说明文档为标准, 只是因为个人的
      能力有限, 进度极为缓慢. 导致语法不一致!
