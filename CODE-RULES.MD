# "IronSQL" Project Code Rules

**About CPP and HPP file name**

- cpp file and hpp file name must use "iron" as prefix

**About namespace**

- namespace must use prefix `Iron`
- private:
  - `_NamespaceName_`
- public:
  - `NamespaceName`

**About class**

- private:
  - `_ClassName`
- public:
  - `ClassName`

**About function**

- private:
  - `_functionName`
- public:
  - `FunctionName`
- espacially:
  - `name_()`
  - `funcName_()`

**About struct**

- private:
  - `StructName_`
- public:
  - `StructName`
