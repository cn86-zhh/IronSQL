# 欢迎来到 "IronSQL" 项目

**项目介绍**

- IronSQL 是一个基于 C++ 语言开发的轻量级别个人数据管理系统。

- IronSQL 是渺小不堪的，它的功能有限，只能满足个人用户的基本需求。
  但是在美学设计上，它借鉴了 MySQL 的格式化输出美学方案，使得数
  据的可视化展示更加直观和美观。

- IronSQL 使用 std::variant 实现真正的类型安全存储，支持 int、
  bigint、maxint、float、double、bool、char、string 等多种数据
  类型。系统会自动识别并转换用户输入的数据为适当的类型，确保
  数据存储的效率和准确性。

- IronSQL 目前有很多不完善的地方，我们希望用户能够及时反馈问题，
  帮助我们改进项目。
