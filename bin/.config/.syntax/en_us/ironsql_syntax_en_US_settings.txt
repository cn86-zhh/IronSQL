[*] 'SETTINGS' Operation Syntax:

[*] Use 'SET' statement to adjust settings:
[*] set -enable --debug --highlight
[*] set -disable --debug --highlight
[*] set -enable --debug
[*] set -disable --debug
[*] set -enable --highlight --for-help

[*] Note!
[*] When using the 'SET' statement to adjust settings, you should pay attention to the following points:
    set -enable --debug --highlight: Enable debug mode and highlight
    set -disable --debug --highlight: Disable debug mode and highlight
    set -enable --debug: Enable debug mode
    set -disable --debug: Disable debug mode
    set -enable --highlight --for-help: Enable highlight for help messages
